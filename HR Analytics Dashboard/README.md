# HR Analytics Dashboard (Tableau)

## Project Overview
This project presents an interactive HR analytics dashboard built using Tableau to analyze workforce distribution, employee demographics, and salary trends. The dashboard enables data-driven decision-making by highlighting key HR metrics and patterns.

---

## Objective
To visualize and analyze employee data in order to identify trends in workforce distribution, salary structure, and employee performance.

---

## Dataset
- HR dataset containing employee details
- Features include: department, age, gender, education, salary, and job roles

---

## Dashboard Features

### Key Performance Indicators (KPIs)
- Total Employees: 7,595  
- Total Hired: 8,498  
- Total Terminated: 903  

### Visualizations
- Department-wise employee distribution  
- Gender distribution (Male vs Female)  
- Education vs Age analysis  
- Education vs Performance comparison  
- Salary distribution by education and gender  
- Age vs Salary (role-based analysis)  
- Location-based employee distribution  

### Interactivity
- Dynamic filtering for better analysis  
- Interactive charts for exploring workforce trends  

---

## Key Insights
- Majority of employees fall within the 25–44 age group  
- Male employees slightly outnumber female employees (~54% vs ~46%)  
- Higher education levels (Master’s, PhD) are associated with higher salaries  
- IT and managerial roles show higher salary ranges compared to other roles  
- Departments like Operations and Sales have the highest employee count
